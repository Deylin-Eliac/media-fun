Guía de Reacciones para Plugins

Aprende a implementar la lógica de reacciones en cualquier comando de tu bot. La clave es seguir 3 pasos.

Paso 1: La Preparación (Importación de Funciones)

En la parte superior de tu archivo de plugin, debes importar las dos funciones de la librería de reacciones. Estas son tus herramientas para comunicarte con la librería.


// Importa las funciones de la librería de reacciones.
// Asegúrate de que la ruta sea correcta según la ubicación de tu plugin.
import { createMessageWithReactions, setActionCallback } from '../lib/reacciones.js';

Nota: ../lib/reacciones.js es la ruta más común. Adáptala si tu archivo de librería está en otro lugar.

Paso 2: La Lógica del Comando (Dentro del handler)

Dentro de tu handler, harás tres cosas: preparar el mensaje, definir las acciones para los emojis, y registrar el mensaje.


let handler = async (m, { conn, text, command }) => {
  // Prepara el mensaje que quieres enviar al usuario.
  const mensajeParaReaccionar = `¡Hola! ¿Qué quieres hacer?\n\n👍 = Opción 1\n👎 = Opción 2`;

  // Define las acciones. Asigna cada emoji a un 'type' único.
  // 'data' puede contener cualquier información que necesites para la acción.
  const actions = {
      '👍': { type: 'opcion_uno', data: { parametro1: 'valorA' } },
      '👎': { type: 'opcion_dos', data: { parametro2: 'valorB' } },
  };

  try {
      // Envía el mensaje y guarda el objeto del mensaje. Esto es CRUCIAL.
      const msg = await conn.sendMessage(m.chat, { text: mensajeParaReaccionar }, { quoted: m });

      // Llama a esta función para decirle a la librería que este mensaje
      // debe escuchar las reacciones definidas en el objeto 'actions'.
      await createMessageWithReactions(conn, msg, actions);

  } catch (error) {
      console.error(error);
      return m.reply('Ocurrió un error al procesar tu solicitud.');
  }
};

Nota: El objeto msg que devuelve conn.sendMessage contiene el ID del mensaje, que la librería necesita para funcionar.

Paso 3: Las Acciones (Fuera del handler)

Ahora tienes que crear las funciones que se ejecutarán cuando se detecten las reacciones. Estas funciones deben ir fuera de tu handler.


// Crea la función para la acción 'opcion_uno'
setActionCallback('opcion_uno', async (conn, chat, data) => {
    // La lógica de esta acción.
    // 'data' contiene el objeto que le pasaste en el handler.
    await conn.sendMessage(chat, { text: `Has elegido la Opción 1. Dato extra: ${data.parametro1}` });
});

// Crea la función para la acción 'opcion_dos'
setActionCallback('opcion_dos', async (conn, chat, data) => {
    // La lógica de esta otra acción.
    await conn.sendMessage(chat, { text: `Has elegido la Opción 2. Dato extra: ${data.parametro2}` });
});

Nota: setActionCallback solo debe llamarse una vez por cada "tipo" de acción.

Resumen Final

Para implementar las reacciones en cualquier plugin:

  1. Importa createMessageWithReactions y setActionCallback.
  2. Define un objeto actions y llama a createMessageWithReactions en tu handler.
  3. Crea las funciones de lógica y regístralas con setActionCallback fuera del handler.

¡Con estos 3 pasos, puedes agregar el sistema de reacciones a cualquier plugin de tu bot de WhatsApp!
