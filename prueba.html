Guía Interactiva: Dominando el Sistema de Reacciones

Esta guía te enseñará, paso a paso, a implementar la lógica de reacciones. Intenta escribir el código tú mismo y luego verifica si lo hiciste bien.

Paso 1: La Preparación

Asegúrate de que tu plugin importe las dos funciones clave de la librería de reacciones.

Mini-Test 1: Intenta escribir la línea de importación. Debe ser la primera línea de tu plugin.
Click para ver la solución 

// Debe ir al inicio del archivo, antes del handler.
import { createMessageWithReactions, setActionCallback } from '../lib/reacciones.js';

Paso 2: La Lógica del Comando (Dentro del handler)

Aquí es donde definimos el plan de acción: qué mensaje se envía y qué hará cada emoji. Este es el corazón de tu comando.

Mini-Test 2: Escribe el código dentro de un handler que envíe un mensaje y registre las acciones para los emojis 👍 y 👎.
Click para ver la solución 

let handler = async (m, { conn }) => {
  const mensajeParaReaccionar = `¡Hola! Prueba las opciones:\n\n👍 = Opción 1\n👎 = Opción 2`;

  const actions = {
      '👍': { type: 'opcion_uno', data: { chat: m.chat } },
      '👎': { type: 'opcion_dos', data: { chat: m.chat } },
  };

  try {
      const msg = await conn.sendMessage(m.chat, { text: mensajeParaReaccionar }, { quoted: m });
      await createMessageWithReactions(conn, msg, actions);
  } catch (error) {
      m.reply('Ocurrió un error.');
  }
};

Paso 3: Las Acciones (Fuera del handler)

Ahora, crea las funciones que se ejecutarán para cada "tipo" de acción que definiste en el paso anterior. Estas funciones contienen la lógica real.

Mini-Test 3: Escribe el código para las dos funciones de `setActionCallback` (opcion_uno y opcion_dos).
Click para ver la solución 

// Lógica para el emoji 👍
setActionCallback('opcion_uno', async (conn, chat, data) => {
    await conn.sendMessage(chat, { text: '¡Elegiste la Opción 1!' });
});

// Lógica para el emoji 👎
setActionCallback('opcion_dos', async (conn, chat, data) => {
    await conn.sendMessage(chat, { text: '¡Elegiste la Opción 2!' });
});

Paso 4: Verificación Final (El Test Completo)

Una vez que hayas escrito todo el código, reinicia tu bot. Ahora, ejecuta estos comandos y verifica los resultados.

Test de Verificación
Acción: Escribe el comando que creaste (por ejemplo, .mando o .prueba).
Resultado Esperado: El bot debe responder con el mensaje que contiene las instrucciones para reaccionar.
Acción: Reacciona con 👍.
Resultado Esperado: El bot debe responder con "¡Elegiste la Opción 1!".
Acción: Reacciona con 👎.
Resultado Esperado: El bot debe responder con "¡Elegiste la Opción 2!".

Si todas las pruebas pasaron, ¡felicidades! Has dominado la implementación del sistema de reacciones. Ahora puedes usar este patrón en cualquier plugin que desees.
